# Linux System Health Script

## Overview
This personal project demonstrates a Bash shell script created to automate system health monitoring on Kali Linux.

## Features
- Displays current date and time
- Shows system uptime
- Checks memory usage
- Checks disk usage
- Fully automated in a single script

## Files Included
- System_Health_Script_Report.docx
- system_health.sh
- Screenshots (if applicable)

## Tools & Environment
- Kali Linux
- Bash Shell
- Linux Terminal

## Learning Outcome
This project helped me understand basic shell scripting, automation of system tasks, and Linux system monitoring. These skills are essential for roles in cybersecurity, system administration, and DevOps.
